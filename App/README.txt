1 Instalirati XAMPP i startati Apache server
2 Instalirati .net core
3 Otvoriti u VSCodeu App i pokrenuti ga

1) HOME
Korisnik može unosom naslova filma u text field i pritiskom na button dobiti detaljne informacije o tom filmu (godinu, žanr, redatelja, jezik, imdb rating i radnju).
Informacije se povlače s OMDB APIJA http://www.omdbapi.com/ u obliku JSON-a
2) CONTACT
Korisnik može unijeti svoje ime, prezime i ime najdražeg filma, pritiskom na button kreira se (ili ukoliko postoji ažurira) XML file.
